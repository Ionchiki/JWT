Запуск проекта

Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Alpha-x-prog/task5.git
   cd task5
```
Перейдите в директорию backend и установите зависимости:
cd backend
npm install
Запустите сервер:
node server.js
Откройте http://localhost:3000/ в браузере, чтобы использовать приложение.
